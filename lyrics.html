Tipos de Overflow

El overflow en CSS se refiere a cómo se maneja el contenido que excede el tamaño de un elemento contenedor.

Existen varios tipos de overflow:

  • visible: El contenido que excede el tamaño del contenedor se muestra fuera de él.
  • hidden: El contenido que excede el tamaño del contenedor se oculta.
  • scroll: Se agrega una barra de desplazamiento para permitir ver el contenido que excede el tamaño del contenedor.
  • auto: Se agrega una barra de desplazamiento solo si es necesario.


Display Property

La propiedad display en CSS se utiliza para controlar cómo se muestra un elemento en la página web.

Existen varios valores para la propiedad display:

  • block: El elemento se muestra como un bloque, ocupando todo el ancho disponible, uno acaba debajo de otro.
  • inline: El elemento se muestra en línea, ocupando solo el espacio necesario.se olvida del widht...
  • inline-block: El elemento se muestra en línea, pero se comporta como un bloque en linea al lado. hay width
  • none: El elemento no se muestra en la página.

Posiciones

Las posiciones pueden ser:

  • static: La posición por defecto, los elementos se posicionan en el flujo normal del documento.
  • relative: La posición se establece en relación a su posición original en el flujo del documento.
  • absolute: La posición se establece en relación al contenedor más cercano con posición relativa.
  • fixed: La posición se establece en relación a la ventana del navegador, no se desplaza al hacer scroll.
  • sticky: Combina las propiedades relative y fixed, el elemento se comporta como relativo hasta que alcanza un cierto punto de desplazamiento, luego se comporta como fijo.
